# Ebook Library API (Node.js + Express + TypeScript)

A simple Ebook Library backend built with Node.js, Express, TypeScript, Prisma, and Neon (PostgreSQL). Includes JWT-based authentication middleware and basic CRUD APIs for managing ebooks.

## Installation

Install with yarn

```bash
  yarn
```
To generate prisma schemas

```bash
  npx prisma generate
```

## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`PORT`

`DATABASE_URL`

`JWT_SECRET`
